Crear una nueva rama de funcionalidad utilizando Git Flow en un proyecto existente. 
Agregar un archivo con un mensaje de bienvenida en la nueva rama. 
Luego, fusionar la rama de funcionalidad con la rama de desarrollo utilizando 
la estrategia de fusión de Git Flow. Por último, eliminar la rama de funcionalidad. 
Se espera que el estudiante utilice los comandos y herramientas de Git Flow para 
completar el ejercicio y que documente adecuadamente el proceso de implementación.
